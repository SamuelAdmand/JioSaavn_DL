## ğŸš€ Getting Started

Follow these steps to get the project running on your local machine.

### Prerequisites

- **Node.js** (v18.x or higher)
- **NPM** or **Yarn**

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/JioSaavn_DL.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd JioSaavn_DL
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Configure Environment:**
   Update the `GEMINI_API_KEY` in [.env.local](.env.local) with your API key.

5. **Fire it up:**
   ```bash
   npm run dev
   ```
